# Информационная система колледжа

Проект разработан в рамках производственной практики по профессиональному модулю ПМ.03 «Ревьюирование программных продуктов».

Цель проекта - разработка информационной системы для автоматизации учета данных образовательного учреждения.

## Технологии

- C# (.NET)
- Windows Forms
- Git + GitHub
- UML
- ER-моделирование

## Функциональные возможности

- учет студентов;
- учет преподавателей;
- учет учебных групп;
- учет дисциплин;
- хранение информации об оценках студентов;
- формирование справочной информации;
- просмотр данных образовательного процесса.

## Требования к окружению

- Windows 10/11
- Visual Studio 2022
- .NET Framework
- Git

## Структура проекта

- README.md - описание проекта;
- TechnicalSpecification.md - техническое задание;
- DatabaseDescription.md - описание базы данных;
- ER_Diagram.md - ER-диаграмма информационной системы.

## Сборка и запуск

1. Открыть проект в Visual Studio.
2. Выполнить сборку проекта.
3. Нажать F5 для запуска приложения.
4. Проверить работу формы «Сотрудник предприятия».
